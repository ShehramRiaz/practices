* Selector which targets HTML elements by it's name is called element selector.

* Universal selector (*) is used to select all elements.
* Universal Selector has 0 specificity.
 
* ID selector is used to select an element of specific ID
* The ID selector starts from # followed by the id of an element.
* The ID should be unique per page.
* It is more specific than element and class selectors.

* Class Selector is used to select multiple elements at once having that class.
* Unlike ID selector, Class selector can be used multiple times within the document.
* Class Selector starts from dot "." followed by class name.
* It is less specific than ID selector and more specific than element selector but we will generally use class selectors more than ID selector.
