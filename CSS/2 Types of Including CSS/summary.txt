* There are three types of including CSS in html document.
- Inline CSS
    This type of styling is usually ignored because the CSS code is mixed with the HTML code.
    This type of styles has the higher precedence than internal and external CSS.
    This is done by using style attribute on specific element.
    That styles are only applied to that specific element on which the style attribute was set.
    
- Internal CSS
    Internal CSS is accomplished by using <style></style> tag usually placed in the head of the document.
    The styles applied to a selector is reflected by all the elements matching that selector.

- External CSS
    External CSS is done by creating a separate file for the css using .css extensions.
    The styles applied to a selector is reflected by all the elements matching that selector.
    The file must be included in the html document before use.
    In case of multiple files using same selector, the last file included or the last style tag declarations wins.
