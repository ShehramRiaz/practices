* The text-transform property does not actually change the text case. Instead, it only change the visuals of the text.
* If we copy that transformed text, The copied text would be the raw text in the HTML document.

* font-family property have a fallback mechanism by allowing us to specify more than one font families.
* Web safe fonts are those that are available and accessible by any browsers and we can safely use that fonts.

* We can use the google fonts by copying the font's link tag.
